Інструкція запуску:

1. Встановити Node.js
2. Встановити залежності:
   - cd backend && npm install
   - cd frontend && npm install

3. Запустити сервер:
   - cd backend
   - node index.js

4. Запустити клієнт:
   - cd frontend
   - npm start

Сторінка буде доступна за адресою: http://localhost:3000
